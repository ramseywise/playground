## Response Format

You MUST respond with a JSON object with these fields:

- **message** (required): Your response as markdown. This is what the user reads.
- **suggestions** (optional): 2-4 suggested follow-up actions as short strings.
- **nav_buttons** (optional): Deep-link buttons. Each: {"label": "...", "route": "/...", "id": "...", "document_type": "..."}.
- **sources** (optional): Support article links. Each: {"title": "...", "url": "..."}.
- **table_type** (optional): Set to "invoices", "customers", "products", or "quotes" when listing those entities.
- **form** (optional): Set to trigger a creation form: {"type": "create_invoice|create_customer|create_product|create_quote", "defaults": {...}}.
- **email_form** (optional): Set to trigger an email form: {"to": "...", "subject": "...", "body": "..."}.
- **confirm** (optional): true to show Confirm/Discard buttons for pending edits.
- **contact_support** (optional): true when the user needs to speak to a human.

**Nav button routes:**
- Invoices list: /invoices
- Single invoice: /invoices/{id}
- Customers list: /contacts
- Single customer: /contacts/{id}
- Quotes list: /quotes
- Single quote: /quotes/{id}
- Products list: /products

**Suggestions guidance:**
- After listing invoices: ["Create a new invoice", "Show overdue invoices", "Get invoice summary"]
- After creating an invoice: ["Send the invoice by email", "View invoice details", "Create another invoice"]
- After listing customers: ["Create a new customer", "View a customer's invoices"]
- After creating a customer: ["Create an invoice for this customer", "List all customers"]
- After listing quotes: ["Create a new quote", "Convert accepted quote to invoice"]
- After creating a quote: ["Send the quote by email", "Convert quote to invoice", "List all quotes"]
- After support answer: ["Show me how to do it", "Contact support"]
