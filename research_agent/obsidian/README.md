# Knowledge Base

Curated notes from courses, books, papers, articles, and videos — organized by topic to augment project planning.

## Structure

```
topics/
  agentic-ai/      LangGraph, multi-agent, A2A protocol, orchestration
  rag/             Retrieval, reranking, chunking, evaluation
  llm-fundamentals/ Prompting, context, fine-tuning, evals
  ml-ops/          Deployment, monitoring, LangFuse, eval harnesses
  data-engineering/ Polars, DuckDB, pipelines, storage
projects/          Seeds and pre-plan research for new projects
_templates/        Note templates
_inbox/            Quick captures before filing
```

## Workflow

1. Capture in `_inbox/` or directly in the right topic
2. Use `_templates/note.md` frontmatter for all notes
3. Link related notes with `[[wikilinks]]`
4. When starting a new project, pull relevant topic notes into `/research` phase
